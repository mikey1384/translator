# v1.13.0

## Title
safer billing, cleaner updates, and reliability improvements

## Release Notes

- Added stronger protection against accidental or abusive credit usage.
- Moved Stage5 authentication to a safer token-based flow.
- Improved app update handling so required updates can be surfaced cleanly.
- Fixed BYO provider selection and credit estimate inconsistencies.
- Simplified dubbing by removing voice cloning.
- Cleaned up video recommendation behavior and clarified YouTube-only search.
